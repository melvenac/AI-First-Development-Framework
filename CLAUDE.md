# Claude Code Entry Point

> **This project uses the AI-First Development Framework.**

All project documentation, tasks, session logs, skills, and workflows live in `.agents/`.

## Getting Started

1. Read `.agents/FRAMEWORK.md` to understand how the framework works
2. Read `.agents/SYSTEM/SUMMARY.md` for current project state
3. Read `.agents/TASKS/INBOX.md` for the task backlog
4. Read `.agents/SYSTEM/RULES.md` for coding standards

## Session Lifecycle

- **Start a session:** Run `/start` (see `.agents/workflows/start.md`)
- **End a session:** Run `/end` (see `.agents/workflows/end.md`)
